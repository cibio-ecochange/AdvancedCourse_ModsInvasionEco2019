NOTES/DESCRIPTION

- This file contains all possible Normalized Difference Combinations (NDI) calculated from the 8 
spectral bands of WorldView-2

NDI = (bi - bj) / (bi + bj)

where bi and bj are two any different bands; bi, bj order does not matter
